My TOP 5 HTML Tags I've Learned So Far

<em> </em>
Emphasis tag: Emphasizes a slightly more important section of text on a page. Screen readers will actually change their voice for this section of text.
<mark> </mark>
Highlighting text tag: Used to highlight a piece of text for reference. For example, the keywords from the recent search results page in the current navigation item.
<blockquote> </blockquote>
Blockquote tag: used to create a large quote which will stand alone from the other text.
<b> </b>
Keyword tag: Highlights a keyword, such as the main subject of an article, in a sentence or a paragraph.
<p> </p>
Paragraph tag: Used for adding an entire paragraph of text. A web browser will add a space before and after the text.
